# 🎤 아이돌 매니저 웹

당신의 선택이 아이돌의 성격과 성장을 결정하는 인터랙티브 육성 시뮬레이션입니다.
실시간 스탯 변화, 채팅, 성장 리포트, 데뷔 카드 생성까지 - 레트로 8-bit 게임 스타일로 즐겨보세요.

---

## ✨ 주요 기능

### 🎯 핵심 경험
1. **온보딩 퀴즈** - 10개의 질문으로 아이돌의 초기 성격 결정
2. **인터랙티브 채팅** - 아이돌과 대화하며 스탯 변화 확인
3. **실시간 스탯** - 친밀도, 다정함, 자신감을 시각적 대시보드로 확인
4. **다이나믹 페르소나** - 5가지 성격 유형이 말투와 반응에 영향
5. **성장 리포트** - 시간에 따른 스탯 변화를 차트로 확인
6. **데뷔 카드** - 아이돌 스탯을 이미지로 저장하고 공유

### 🎨 레트로 8-bit 디자인
- Windows 95/98 스타일 윈도우 UI
- 픽셀 폰트 (Press Start 2P, VT323)
- 10칸 세그먼트 게이지 + 픽셀 하트
- 스캔라인 효과 및 글로우 애니메이션
- 진한 파란색 그리드 배경

**사전 요구사항:**
- Ollama 설치 및 실행
- 백엔드 서버 (`idol-manager-backend`)

**설정:**

```bash

ollama serve

# 터미널 2: 백엔드
cd ../idol-manager-backend
source .venv/bin/activate
uvicorn app.main:app --reload

# 터미널 3: 프론트엔드
npm run dev
```

## 🎭 페르소나 시스템

| 페르소나 | 성향 | 말투 |
|---------|------|------|
| 🌟 **다정한 리더** | 다정함↑ 자신감↑ | 따뜻하고 당당함 |
| 🌻 **햇살 같은 동료** | 다정함↑ 자신감↓ | 밝고 겸손함 |
| 🔥 **독기 있는 연습생** | 다정함↓ 자신감↑ | 도전적이고 열정적 |
| 🌙 **조용한 프로** | 다정함↓ 자신감↓ | 차분하고 프로페셔널 |
| ⚖️ **균형잡힌 아이돌** | 중립 | 안정적이고 성숙함 |

**친밀도별 말투 변화:**
- 0~30: 존댓말 (거리감)
- 31~60: 반존댓말
- 61~100: 반말 (친근함)
