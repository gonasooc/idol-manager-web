## 2. 디자인 컨셉: Retro 8-bit Game Style

### 2.1 디자인 방향
**Windows 95/98 + 8-bit RPG 게임**의 융합 스타일
- 노스탤지어를 자극하는 레트로 게임 매니지먼트 시뮬레이션 느낌
- 픽셀 폰트, 청키한 테두리, 세그먼트 게이지로 독특한 개성 표현

### 2.2 핵심 디자인 요소

**폰트**
- 제목: "Press Start 2P" (픽셀 폰트)
- 본문: "VT323" (레트로 모노스페이스)

**색상 팔레트**
```
배경: #1a1a4e (진한 파란색) + 그리드 패턴
창 배경: #f5f5dc (크림색)
강조색: #40e0d0 (청록), #ffd700 (금색), #ff69b4 (핑크)
테두리: Win95 스타일 3D 베벨 (흰색/회색)
```

**UI 컴포넌트**
| 요소 | 스타일 |
|------|--------|
| 창(Window) | Win95 타이틀바 + 최소화/닫기 버튼 |
| 버튼 | 픽셀 테두리 + 눌림 효과 |
| 게이지 | 10칸 세그먼트 + 픽셀 하트 아이콘 |
| 말풍선 | 레트로 픽셀 스타일 |
| 카드 | ASCII 스탯바 + 스캔라인 효과 |

**애니메이션**
- 스탯 변화: 글로우 + 픽셀 바운스
- 레벨업: 깜빡임 + 별 파티클
- 텍스트: 타이핑 효과

## 3. 핵심 데이터 및 UI 요소 (Visible Stats)

### 3.1 실시간 상태 바 (STATUS.exe)
- 토글 가능한 Win95 스타일 윈도우
- 최소화 시 타이틀바에 요약 정보 표시

**표시 항목:**
- 친밀도 (Bond Level): 0~100, 픽셀 하트 + 10칸 세그먼트 게이지
- 성향 지수: 다정함↔냉정함 / 소심함↔자신감 레트로 슬라이더
- 페르소나 타이틀: 이모지 + 칭호

### 3.2 수치 변화 애니메이션
- 플로팅 텍스트: +5, -2 글로우 효과
- 구간 돌파(30/60/90): 게이지 색상 변화 + 반짝임
